run_agent_test

 run_agent_test (agent:str, scenario:str,
                 project_path:Optional[pathlib.Path]=None,
                 timeout:int=120,
                 save_session:Optional[pathlib.Path]=None)

Run an automated agent test.

Creates a test project, invokes the agent with a scenario prompt, records tool calls, and validates against expected behavior.

Args: agent: Which agent CLI to use (“claude” or “codex”) scenario: Test scenario name (edit_py_file, fix_lint, etc.) project_path: Optional existing project to test (creates temp if None) timeout: Max seconds to wait for agent save_session: Optional path to save session JSON

Returns: Dict with test results including passed, failures, stats

Type Default Details
agent str “claude” or “codex”
scenario str
project_path Optional None
timeout int 120
save_session Optional None
Returns Dict
