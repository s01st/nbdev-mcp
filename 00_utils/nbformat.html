has_standard_ending

 has_standard_ending (nb:nbformat.notebooknode.NotebookNode)

Check if notebook has the standard ending structure.

Standard ending is: - ## Next (markdown) - #| export (empty code cell) - ## Export (markdown) - #| hide + [nbdev_export](https://s01st.github.io/nbdev-mcp/11_tools/nbdev.html#nbdev_export) (code cell)

Type Details
nb NotebookNode The notebook to check.
Returns bool True if notebook has the standard ending.

find_insertion_point

 find_insertion_point (nb:nbformat.notebooknode.NotebookNode,
                       cell_type:str='code')

Find the best insertion point for a new cell.

For code cells: - If notebook has standard ending, insert before the ‘## Next’ section - Otherwise insert before ‘## Export’ if it exists - Otherwise append to end

For import cells: - Insert after the Imports section header

Type Default Details
nb NotebookNode The notebook.
cell_type str code Type of cell being inserted: ‘code’, ‘markdown’, or ‘import’.
Returns int The index where the new cell should be inserted.

Insert a cell into a notebook with smart positioning.

Type Default Details
