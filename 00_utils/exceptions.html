Exceptions

Custom exception classes with notebook location tracking

Imports

Notebook Location

Track where an error/warning originated in the source notebooks.

source

NotebookLocation

 NotebookLocation (notebook:str, cell:int=-1, line:int=0)

Track the source location of an error within nbdev notebooks.

Args: notebook: Name or path of the source notebook (e.g., ‘03_utils.ipynb’). cell: Zero-based cell index where the error occurred. line: One-based line number within the cell (default 0 = unknown).

Base Exception

source

NotebookException

 NotebookException (message:str,
                    location:Optional[__main__.NotebookLocation]=None)

Base exception for nbdev-mcp with optional notebook location tracking.

Args: message: The error message. location: Optional source location in the notebooks where the error originated.

ProjectNotFoundError

 ProjectNotFoundError (selector:str)

Raised when an nbdev project cannot be found or resolved.

Args: selector: The selector/path that failed to resolve.

source

NotebookNotFoundError

 NotebookNotFoundError (path:Any)

Raised when a notebook file cannot be found.

Args: path: The notebook path that was not found.

source

CellNotFoundError

 CellNotFoundError (notebook:str, cell_index:int, total_cells:int)

Raised when a cell index is out of range.

Args: notebook: The notebook name. cell_index: The invalid cell index. total_cells: The actual number of cells in the notebook.

source

ExportError

 ExportError (message:str,
              location:Optional[__main__.NotebookLocation]=None)

Raised when nbdev_export fails.

Args: message: Error details from the export command. location: Source location if known.

source

LintError

 LintError (rule:str, message:str,
            location:Optional[__main__.NotebookLocation]=None)

Raised when linting finds a blocking issue.

Args: rule: The lint rule that was violated. message: Description of the violation. location: Source location of the violation.

warn_with_location

 warn_with_location (message:str, notebook:Optional[str]=None,
                     cell:int=-1, line:int=0, category:type=<class
                     '__main__.NbdevMcpWarning'>)

Issue a warning with notebook location context.

Args: message: The warning message. notebook: Name of the source notebook. cell: Zero-based cell index (default -1 = unknown). line: One-based line number within cell (default 0 = unknown). category: Warning category class (default NbdevMcpWarning).

source

raise_with_location

 raise_with_location (message:str, notebook:Optional[str]=None,
                      cell:int=-1, line:int=0, exc_class:type=<class
                      '__main__.NotebookException'>)

Raise an exception with notebook location context.

Args: message: The error message. notebook: Name of the source notebook. cell: Zero-based cell index (default -1 = unknown). line: One-based line number within cell (default 0 = unknown). exc_class: Exception class to raise (default NotebookException).

Raises: NotebookException: Or subclass specified by exc_class.
