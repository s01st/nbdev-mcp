depgraph

Hierarchical dependency graph with symbol-level tracking and duplication detection

Imports

Data Model

source

SymbolNode

 SymbolNode (id:str, name:str, kind:str, notebook:str, module:str,
             cell_index:int, source:str='', source_hash:str='',
             line_count:int=0)

A symbol in the dependency graph.

Type Default Details
id str Unique identifier (module.notebook.symbol).
name str Symbol name.
kind str Type: ‘function’, ‘class’, ‘constant’, ‘type’.
notebook str Source notebook path.
module str Target module name.
cell_index int Cell index in notebook.
source str Source code.
source_hash str Hash of normalized source for duplication detection.
line_count int 0 Number of lines of code.

source

NotebookNode

 NotebookNode (id:str, path:pathlib.Path, module:str,
               symbols:List[__main__.SymbolNode]=<factory>)

A notebook in the dependency graph.

Type Default Details
id str Unique identifier (relative path).
path Path Full path to notebook.
module str Target module (from default_exp).
symbols List Symbols defined in this notebook.

source

ModuleNode

 ModuleNode (id:str, notebooks:List[__main__.NotebookNode]=<factory>)

A module (package) in the dependency graph.

Type Default Details
id str Module name.
notebooks List Notebooks that export to this module.

source

Edge

 Edge (source:str, target:str, kind:str='import', weight:float=1.0,
       metadata:Dict[str,Any]=<factory>)

A dependency edge in the graph.

Type Default Details
source str Source node ID (the supporter/dependency).
target str Target node ID (the dependent).
kind str import Edge type: ‘import’, ‘usage’, ‘duplicate_exact’, ‘duplicate_similar’, ‘duplicate_suspected’.
weight float 1.0 Edge weight (1.0 for imports, similarity score for duplicates).
metadata Dict Additional edge metadata.

source

DependencyGraph

 DependencyGraph (project:pathlib.Path, lib_name:str,
                  modules:Dict[str,__main__.ModuleNode]=<factory>,
                  notebooks:Dict[str,__main__.NotebookNode]=<factory>,
                  symbols:Dict[str,__main__.SymbolNode]=<factory>,
                  edges:List[__main__.Edge]=<factory>)

Complete hierarchical dependency graph.

Type Default Details
project Path Project root path.
lib_name str Library name.
modules Dict Module nodes by ID.
notebooks Dict Notebook nodes by ID.
symbols Dict Symbol nodes by ID.
edges List All edges in the graph.

source

detect_duplicates

 detect_duplicates (graph:__main__.DependencyGraph,
                    similarity_threshold:float=0.8)

Detect potential code duplications in the graph.

Type Default Details
graph DependencyGraph The dependency graph to analyze.
similarity_threshold float 0.8 Minimum similarity ratio for ‘similar’ duplicates (0-1).
Returns List Duplication edges (exact, similar, suspected).

Interactive Visualization

source

generate_interactive_graph

 generate_interactive_graph (project:Optional[str]=None,
                             output_path:Optional[str]=None,
                             include_duplicates:bool=True,
                             similarity_threshold:float=0.8)

Generate an interactive dependency graph visualization.

Uses D3 circle packing to show hierarchical structure: - Modules contain notebooks contain symbols - Click to zoom into any level - Edges show import dependencies and code duplications

Type Default Details
project Optional None Project path or alias.
output_path Optional None Output HTML file path. Defaults to project/dep_graph.html.
include_duplicates bool True Whether to detect and show code duplications.
similarity_threshold float 0.8 Threshold for similar duplicate detection.
Returns Dict Result with graph stats and output path.
