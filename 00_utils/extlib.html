generate_modidx

 generate_modidx (package_name:str, doc_base_url:Optional[str]=None,
                  max_depth:int=2)

Generate a module index for an installed package.

Inspects the package to discover public symbols and creates a modidx-compatible dictionary.

Type Default Details
package_name str The package to index.
doc_base_url Optional None Base URL for documentation. Uses KNOWN_DOC_URLS if not specified.
max_depth int 2 Maximum submodule depth to scan.
Returns Dict A modidx-compatible dictionary.
